# 📚 Code Sheets - Major Languages & Related Resources (English)
Welcome to my central repository where I share all my courses, code sheets, and useful resources for major programming languages and related topics.

# 📖 About
This repository is designed to be a personal and public knowledge base. It contains summaries, code snippets, best practices, and key concepts for various languages and technologies. Whether you are a beginner looking to learn the basics or an experienced developer seeking a quick reference, I hope you find valuable information here.

# 📂 Repository Structure
The repository is organized by language/technology. Each main folder contains dedicated subfolders or files.

text
/
├── python/             # Python cheat sheets (basics, libraries, etc.)
├── java/               # Java cheat sheets (OOP, collections, etc.)
├── git/                # Git cheat sheets (commands, workflows)
├── Rust/
└── ...

(Feel free to explore and suggest improvements or additions!)

# 🚀 How to Use These Sheets
Navigation: Browse the folders to find the language or topic you are interested in.

Consultation: Open the .md (Markdown) files or code files to see explanations and examples.

Active Learning: Read the concepts, then try the code snippets yourself in your development environment.

Quick Reference: Use the sheets as a cheatsheet while coding.

# ✨ Typical Content of a Sheet
Each sheet may include:

An introduction to the concept.

Commented code snippets.

Common use cases.

Pitfalls to avoid.

Links to official documentation or other resources.
